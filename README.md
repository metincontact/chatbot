# 🤖 Chatbot

Simple chatbot application built with React.

## 🚀 Live Demo

https://metin-chatbot.vercel.app

## ✨ Features

* Send and receive messages
* User & bot message UI
* Auto scroll

## 🛠️ Tech

* React (Vite)
* JavaScript
* CSS

## ▶️ Run Locally

```bash
git clone https://github.com/metincontact/chatbot.git
cd chatbot
npm install
npm run dev
```
